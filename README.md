# Trabalho-Final-1-ano-faculdade-CC
Este é meu ultimo trabalho do primeiro ano do curso ciencias da computação

Titulo Quiosque de atendimento do DOG MANIA

o codigo apresenta 3 botões inicias sendo

o Primeiro é Comprar lanches que te mostra uma tela com as opções de adicionar xis salada, coca e pudim
após selecionar todos os itens desejados é possivel concluir pedido aonde ele ira para a lista de atendimento
onde todos os pedidos se encontram.

Segundo botão, Ver atendimentos ele te leva para a lista de atendimento onde é possivel ver todos os pedidos
sem ter a opção de editar apenas ver.

Terceiro botão e a ultima é Gerenciar estabelecimento nele alem de ter acesso aos atendimentos é possivel alterar
eles para Pronto, Em Preparo e Pronto modificando todas as que desejar e logo após selecionar o status desejado é
necessario apertar o botão escrito "Atualizar Status" para editar o pedido.

agradeço a compreensão tenha um otimo dia.
